# Personal Portfolio Website

This project is a personal portfolio website developed for my Computer Science studies. It presents my skills, education, and projects, and includes a simple blog system.

## Features

* Home, Education, Skills, Portfolio, and Contact pages
* Navigation bar
* Blog system with database

## Technologies

* HTML, CSS
* PHP
* MySQL (MAMP)

## How to Run

1. Place the project in `/htdocs/`
2. Start MAMP
3. Open: `http://localhost/hajar-phase2/`
4. Import the database file in phpMyAdmin
